# Cherri
- I developed this Food Delivery Application using Pyqt5 and tkinter,Used Mysql database as backend
# Modules
- Tkinter
- Pillow
- Mysql.connector
- sys
- csv
- PyQt5
# To Run
- User can simply install app using setup file and run Main.exe file
- Or Using Main.py file
